# Contracts: TitanForge Gym Landing Page

This landing page is a fully static frontend with zero backend, zero API
endpoints, and zero external service integrations for v1.

**No API contracts generated.**

All data (pricing, programs, trainers, testimonials, FAQs, benefits, nav links,
footer content) is defined as typed constants in `lib/constants.ts`.

Future contracts (if a backend is added for membership signup, form
submissions, or analytics) would be documented here.
